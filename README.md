# dangke-question

## 项目简介

dangke-question 是一个用于党课学习与题目练习的网页项目，主要用于整理党课相关题库、展示学习资料，并提供便捷的在线浏览与使用入口。项目支持通过 GitHub Pages 部署，适合用于课程学习、培训复习或资料共享。

## 主要功能

- 党课题目与学习资料整理
- 支持网页端在线浏览
- 提供使用指南文档
- 适合个人学习、班级共享或培训复习使用
- 可通过 GitHub Pages 进行静态网页部署

## 项目文件说明

| 文件名称 | 说明 |
|---|---|
| `README.md` | 项目介绍文件 |
| `使用指南.md` | Markdown 格式的使用说明 |
| `使用指南.docx` | Word 格式的使用说明 |
| 其他源码文件 | 网页项目运行所需文件 |

## 使用方法

1. 打开项目在线访问链接；
2. 根据页面提示浏览题目或学习资料；
3. 如需查看详细说明，可阅读 `使用指南.md` 或 `使用指南.docx`；
4. 如需二次修改，可下载或克隆本仓库后编辑相关文件。
